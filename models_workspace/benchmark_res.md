# PureEdgeVLM 端侧推理测速对比表

> 测速基线（Baseline）：骁龙 865，纯 CPU 推理（NCNN 关闭 Vulkan；大模型 llama.cpp n_threads 见各表）。后续接入 GPU / Vulkan 后的加速效果将在此基础上对比。
> 数值为每次模型单独推理的延迟（毫秒），首跑为 warmup 不计入；fps = 1000 / 平均延迟。
> 分辨率（resolution）为输入位图边长（像素），本次在 224 / 640 / 960 三档下各测一遍；输入越大，OCR 等直接处理原图的模型延迟越高。
